# Biomedical-Engineering-Club
![Biomedical Engineering Club](Assets/Images/logo.jpg)


The Biomedical Engineering Club was created to give students an opportunity to explore the intersection of biology, medicine, and engineering through hands-on learning and collaboration. The club aims to make biomedical engineering more accessible to students by providing a space where members can learn about emerging technologies, develop technical skills, and work together on projects that connect engineering concepts to real-world challenges in healthcare and medicine. Through this website, the club can share its work, resources, projects, and opportunities with both current and prospective members, regardless of geographical disparities.

## Repository Overview

This repository contains the source files for the Biomedical Engineering Club website. For organizational purposes, the repository has been simplified and reorganized to make the file structure easier to navigate and understand.

The repository is primarily divided into two main folders:

* **`assets/`** — Contains the website's shared visual assets, including:

  * **`images/`** — Images used throughout the website
  * **`favicon/`** — Website favicon file
  * **`background/`** — Background image used throughout the site

* **`pages/`** — Contains the files for each individual page of the website. Each page is organized into its own folder to keep its associated files together.

> **Note:** The file structure shown in this repository has been simplified for readability and organizational purposes. As a result, some of the file paths and names shown here may not exactly match the paths used by the live website. The actual website may use different file paths and directory structures to ensure that all resources load correctly when the site is deployed. Additionally, the website's index file is referred to as home in this simplified repository structure.

## Shared Files

The website also contains files that apply across multiple pages:

* **`styles.css`** — Contains the shared CSS styling used throughout the website. This file controls the overall visual appearance, layout, typography, and other common design elements across the pages.

* **`homescript.js`** — Contains JavaScript functionality that applies across all pages of the website, including shared interactive features and behaviors.

These shared files allow the website to maintain a consistent design and functionality across its different pages while avoiding unnecessary duplication of code.
